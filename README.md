# Job Tracker Application

A full-stack Job Tracker application built using React.js and Material UI to help users manage and track job applications through different stages of the hiring process.

## Live Demo

Frontend: https://YOUR-VERCEL-URL.vercel.app

Backend API: https://job-tracker-api-hbui.onrender.com/jobs

## Features

* Add, edit, and delete job applications
* Track application status (Applied, Interview, Selected, Rejected)
* Search jobs by company name
* Filter jobs by application status
* Confirmation dialog before deleting jobs
* Snackbar notifications for user actions
* Loading and error state handling
* Responsive UI built with Material UI
* Deployed frontend and backend

## Tech Stack

### Frontend

* React.js
* JavaScript (ES6+)
* Material UI
* Axios
* Vite

### Backend

* JSON Server
* REST APIs

### Deployment

* Vercel (Frontend)
* Render (Backend)
